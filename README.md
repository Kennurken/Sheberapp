# Sheber.kz

Маркетплейс мастеров (сантехника, электрика и др.) для Казахстана: **Flutter**‑клиент и **PHP**‑API.

## Структура репозитория

| Каталог | Назначение |
|--------|------------|
| `sheber_app/` | Мобильное приложение (Dart/Flutter) |
| `htdocs/` | Бэкенд: `init.php`, `api/*.php`, SQL-миграции |
| `deploy/` | Скрипты деплоя, примеры Nginx |

Документация для разработки: **`CLAUDE.md`**, операционные детали — **`SHEBER_CURSOR_HANDOFF.md`**, расширенный контекст — **`SHEBER_CONTEXT.md`**.

## Быстрый старт

**Бэкенд (локально):**

```bash
cd htdocs
# Скопируйте config.local.example.php → config.local.php и заполните БД
php -S localhost:8000 -t .
```

**Клиент:**

```bash
cd sheber_app
flutter pub get
# Добавьте google-services.json в android/app/ (Firebase Console)
flutter run
```
